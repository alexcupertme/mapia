# Contributing

Contributions are welcome! Feel free to open an issue or submit a pull request for bug fixes, new features, or documentation updates.

If you spot something missing in these docs, copy the repository and edit the Markdown under `docs/`, then run `pnpm run docs:build` to verify the site builds cleanly before opening a pull request.
